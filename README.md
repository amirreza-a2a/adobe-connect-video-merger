
# 🚀 Adobe Connect Video Merger 

این پروژه شامل مجموعه‌ای از اسکریپت‌های Bash و Python است که برای خودکارسازی فرآیند **اکسترکت، همگام‌سازی (Sync) و ترکیب (Merge)** ویدیوها و فایل‌های صوتی تکه‌تکه شده در خروجی‌های زیپ Adobe Connect طراحی شده است.

به کمک این ابزار، دیگر نیازی به بررسی دستی تکه‌های صدا (`cameraVoip`) و ویدیوی اصلی ندارید؛ اسکریپت با خواندن متادیتاهای XML، اختلاف زمانی دقیق هر فایل را محاسبه کرده و با FFmpeg آن‌ها را ترکیب می‌کند.

---

## 🛠 پیش‌نیازمندی‌ها

برای اجرای بدون نقص اسکریپت‌ها، ابزارهای زیر باید روی سیستم (ترجیحاً لینوکس/اوبونتو) نصب باشند:

* **FFmpeg**: برای میکس صدا و تصویر بدون افت کیفیت.
* **Unzip**: برای اکسترکت خودکار فایل‌های فشرده کلاس‌ها.
* **Python 3**: برای اجرای اسکریپت پردازش زمان‌بندی‌ها.

نصب ابزارهای مورد نیاز در اوبونتو:

```bash
sudo apt update
sudo apt install ffmpeg unzip python3

```

## 📂 ساختار دایرکتوری‌ها

اسکریپت‌ها به صورت پیش‌فرض بر اساس ساختار پوشه‌ای زیر در دایرکتوری Home کار می‌کنند:

```text
├── 📂 Downloads/
│   ├── 📂 adobeZip/      # محل قرارگیری فایل‌های زیپ دانلود شده (شروع با نام p)
│   └── 📂 adobeVideo/    # محل ذخیره نهایی ویدیوهای ترکیب‌شده (class_01.mkv و...)
└── 📂 Scripts/
    └── mergeVideoAdobe.py # اسکریپت پایتون پردازشگر متادیتا

```

> ⚠️ **نکته درباره Python Shebang:** اسکریپت پایتون به صورت پیش‌فرض از محیط مجازی در مسیر `~/Scripts/.scriptsENV/bin/python3` استفاده می‌کند. در صورت نیاز می‌توانید خط اول فایل `mergeVideoAdobe.py` را به مفسر پایتون خود تغییر دهید.

## ⚙️ نحوه کارکرد سیستم

### ۱. اسکریپت پایتون (`mergeVideoAdobe.py`)

این اسکریپت مغز متفکر همگام‌سازی است:

* فایل تصویر اصلی (`screenshare` یا `mainstream`) و تمام ترک‌های صوتی (`cameraVoip`) را پیدا می‌کند.
* متادیتاهای XML متناظر با هر فایل را پارس کرده و زمان دقیق شروع هر کدام را استخراج می‌کند.
* نقطه مبدأ (اولین زمان ثبت شده) را پیدا کرده و میزان تاخیر (Offset) هر ترک را برحسب میلی‌ثانیه محاسبه می‌کند.
* در نهایت یک دستور پیچیده `ffmpeg` همراه با فیلترهای `adelay` و `amix` می‌سازد تا همه‌چیز کاملاً سینک و در قالب یک فایل `final_class_synced.mkv` لود شود.

### ۲. اسکریپت بش (`process_adobe.sh`)

این اسکریپت مدیریت پوشه‌ها و اتوماسیون انبوه را انجام می‌دهد:

* فایل‌های زیپ موجود در `~/Downloads/adobeZip` را بر اساس زمان دانلود/اصلاح (از قدیمی به جدید) مرتب می‌کند.
* تک‌تک فایل‌ها را به صورت موقت اکسترکت می‌کند.
* اسکریپت پایتون را درون پوشه اجرا می‌کند.
* فایل نهایی خروجی را نام‌گذاری منظم کرده (`class_01.mkv`, `class_02.mkv` و...) و به پوشه `adobeVideo` می‌فرستد و در آخر محیط را پاک‌سازی می‌کند.

## 🚀 راهنمای راه‌اندازی و اجرا

### ۱. دسترسی اجرای اسکریپت‌ها

ابتدا باید به هر دو اسکریپت مجوز اجرا (Executable) بدهید:

```bash
chmod +x process_adobe.sh
chmod +x mergeVideoAdobe.py

```

### ۲. قرار دادن اسکریپت پایتون در PATH (اختیاری اما توصیه شده)

برای اینکه اسکریپت بش بتواند راحت‌تر فایل پایتون را پیدا کند، بهتر است فایل `mergeVideoAdobe.py` را در مسیر اسکریپت‌های سیستم خود یا پوشه‌ای که در `$PATH` تعریف شده (مثل `~/Scripts/` یا `/usr/local/bin/`) قرار دهید.

### ۳. شروع فرآیند ترکیب

کافیست فایل‌های زیپ کلاس‌های ادوبی کانکت را در پوشه `~/Downloads/adobeZip` بریزید و اسکریپت اصلی را اجرا کنید:

```bash
./process_adobe.sh

```

اسکریپت شروع به پردازش تک‌تک فایل‌ها کرده و لاگ مراحل را در ترمینال نمایش می‌دهد. پس از اتمام کار، ویدیوهای نهایی و سینک شده در پوشه `~/Downloads/adobeVideo` آماده تماشا هستند.
